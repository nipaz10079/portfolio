Snoop Alert

Security v1.0.0

About the App

Snoop Alert is an advanced Android security application designed to protect your privacy in public spaces. It uses **Google ML Kit's Face Detection** to constantly monitor the environment through the front camera. If it detects a face looking over your shoulder, it instantly triggers a 'Red Alert' screen to block content visibility.

Key Features

  • Background Protection: Runs as a foreground service to protect you even when using other apps.
  • Offline AI: All face detection happens on-device. No images are ever sent to the cloud.
  • Instant Trigger: Uses CameraX for low-latency video analysis.

CalorieSense

AI Web App

About the Project

A sophisticated web application that automates nutritional tracking using Computer Vision. Users can upload or capture meal photos to receive instant caloric breakdowns, macronutrient analysis, and personalized dietary advice based on their unique health goals.

Key Features

  • Multimodal AI Vision: Powered by Google Gemini API to identify complex food items and estimate portions from pixels.
  • NutriGuide Context Engine: Real-time 'suitability verdict' based on user goals (e.g., Keto, Low Sodium).
  • Neural Scanning UI: Premium Glassmorphism design with a live hardware camera HUD.
  • Persistent Timeline: Saves full nutritional reports locally without re-calling the API.
